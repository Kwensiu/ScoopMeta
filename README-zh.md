# Pailer

[English](README.md) | 简体中文

这是一个基于[AmarBego Pailer](https://github.com/AmarBego/Pailer) 的分支项目，Pailer 是一个开源的 Scoop GUI 管理器

该项目添加了一些上游尚未实现的额外功能和修改

## 安装

[Releases](https://github.com/Kwensiu/Pailer/releases) 页面下载最新版本

## 功能

- 保留了 Pailer 的所有原有功能
- 添加了国际化支持、便携终端、配置文件编辑器等新功能
- 优化了 UI 和用户体验
