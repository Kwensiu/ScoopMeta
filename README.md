# ScoopMeta

English | [简体中文](README-zh.md)

This is a fork of [AmarBego's Rscoop](https://github.com/AmarBego/Rscoop), an open-source GUI manager for Scoop.

It adds some additional features and modifications not yet implemented upstream.

## Installation

Download the latest release from the [Releases](https://github.com/Kwensiu/ScoopMeta/releases) page.

## Features

- Retains all original features of Rscoop
- Adds new features like internationalization support, portable terminal, configuration editor
- Optimizes UI and user experience
